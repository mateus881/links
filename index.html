introdução ao MVC

O padrão MVC (model-view-controler) é uma arquitetura de softwere utilizada no desenvolvimento de aplicações
para separar a lógica de négocios , a interface do usuario e o controle dos dados .

Componentes do MVC

  • Model (modelo)- representa os dados e a lógica de negócios da aplicação .
  • View (Visão)- Responsável pela interface com o usuário e exibição dos dados.
  • Controller (Controlador)- intermedia a comunicação entre model e View
